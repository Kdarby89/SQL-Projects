Dataset cleaning and EDA in MYSQL

This MYSQL code removes duplicates by using:
    • CTE
    • Partitions
    • Creating tables
    • Inserting and deleting data. 
The data is then standardized by using:
    • Trim
    • Updating and altering tables
    • Is and is not null
    • Joins
EDA is performed by using:
    • Min, Max, Sum, ETC
    • Group by
    • Order by
    • Sub string
    • With clause
